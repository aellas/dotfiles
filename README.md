# dotfiles
 My dotfiles for openSUSE / hyprland
 
 - Distro: openSUSE
 - WM: Hyprland
 - Shell: Bash + oh-my-bash
 - Terminal: Kitty
 - Panel: Waybar
 - Launcher: Wofi
 - File Manager: Nemo
 - Screenshots: grimm + slurp
 - Wallpaper: Swaybg

# Noteable Keybinds

| Keybinds  | Action |
| ------------- | ------------- |
| Mod + Q  | Kill Window |
| Mod + ENTER  | Launches Kitty |
| Mod + BACKSLASH | Launches neovim  |
| Mod + SPACE  | Launches Wofi |
| Mod + SHIFT + W  | Restarts Waybar|
