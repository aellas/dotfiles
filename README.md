# dotfiles
 My dotfiles for hyprland
 
 - Distro: [CachyOS](https://cachyos.org/)
 - WM: [Hyprland](https://hyprland.org/)
 - Shell: Fish + [oh-my-fish](https://github.com/oh-my-fish/oh-my-fish)
 - Terminal: Kitty
 - Panel: Waybar
 - Launcher: Wofi
 - File Manager: Nemo
 - Screenshots: Hyprshot
 - Wallpaper: Swaybg

# Noteable Keybinds

| Keybinds  | Action |
| ------------- | ------------- |
| Mod + Q  | Kill Window |
| Mod + ENTER  | Launches Kitty |
| Mod + SPACE  | Launches Wofi |
| Mod + SHIFT + W  | Restarts Waybar|
| Mod + Shift + I  | Set corsair aio to quiet |
| Mod + Shift + O | Set corsair aio to balanced  |
| Mod + Shift + P  | Set corsair aio to extreme  |

> [!NOTE]
> Corsair aio keybinds require liquidctl & [aio.sh](https://github.com/aellas/dotfiles/blob/main/aio.sh) to be placed in /opt/aio!
